# 데일리 인사이트 대시보드 사용 설명서

데일리 인사이트 대시보드는 네이버 메일 등 이메일을 자동으로 수집하고, AI(Gemini)를 사용해 내용을 요약 및 분류하여 시각적인 대시보드로 보여주는 파이썬 기반 도구입니다.

## 1. 사전 준비 사항
- **Python 3.9 이상** 설치 필요
- **Google Gemini API Key** 발급 필요

## 2. 네이버 메일 설정 (IMAP 및 앱 비밀번호)
이 대시보드가 네이버 메일에 접근하려면 2가지 메일 계정 설정이 선행되어야 합니다.

### IMAP 활성화
1. 네이버 메일 접속 후 좌측 하단 톱니바퀴 아이콘(환경설정) 클릭
2. 상단 메뉴 중 **[POP3/IMAP 설정]** 탭 클릭
3. **IMAP/SMTP 설정** 항목에서 '사용함'으로 체크 후 확인

### 2단계 인증 및 애플리케이션 비밀번호 생성 (필수)
네이버 2단계 인증을 사용 중인 경우, 원래 쓰던 로그인 비밀번호로는 외부 프로그램에서 접속할 수 없습니다. 대신 '애플리케이션(앱) 비밀번호'를 새로 생성해야 합니다.
1. [네이버 보안설정 페이지](https://nid.naver.com/user2/help/myInfo?m=viewSecurity)로 접속
2. **2단계 인증** 항목 우측의 **관리** 버튼 클릭
3. 하단 **애플리케이션 비밀번호 관리**에서 종류를 '기타'로 선택, 이름 입력 후 **[생성]** 클릭
4. 임의로 발급되는 영문 형태의 비밀번호(12자리 내외)를 띄어쓰기 없이 복사하여 잘 보관합니다.

> 💡 **특정 메일함 설정하기 (선택)**
> 모든 메일을 읽어오는 대신 정보성 뉴스레터나 중요 메일만 분류하려면, 네이버에서 새 메일함(예: `VIP_In`)을 만들고 네이버 메일 자동 분류 필터로 메일들이 그쪽으로 들어가게 설정하세요. 이후 `config.json`에서 폴더 이름을 지정해두면 해당 폴더만 분석합니다.

## 3. 설정 파일 (`config.json`) 생성 및 작성
폴더 가장 바깥 경로(`daily_insight_dashboard/`)에 `config.json` 이라는 이름의 새 파일을 만들고 아래 코드를 복사해서 붙여넣습니다. 그리고 본인의 정보에 맞게 수정합니다.

```json
{
    "imap": {
        "server": "imap.naver.com",
        "port": 993,
        "email": "본인의네이버아이디@naver.com",
        "password": "위에서_발급받은_앱_비밀번호_입력"
    },
    "gemini": {
        "api_key": "발급받은_Gemini_API_Key_입력",
        "model": "gemini-flash-latest"
    },
    "preferences": {
        "vip_mode": "folder",
        "vip_folder_name": "VIP_In",
        "max_emails_per_day": 10
    }
}
```
* **주의**: `password` 항목에 일반 로그인 비밀번호를 입력하면 작동하지 않습니다.

## 4. 어플리케이션 실행
모든 준비가 끝났다면 터미널(Mac)이나 명령 프롬프트(Windows)를 열고 프로젝트 폴더 경로로 이동한 뒤, 아래 명령어를 입력하여 실행합니다.

```bash
python run.py
```

`run.py` 를 실행하면 아래 작업이 순서대로 자동 진행됩니다.
1. 프로그램 구동에 필요한 라이브러리(`requirements.txt`) 자동 설치
2. 이메일 서버에 접속하여 최신 메일 수집
3. Gemini AI 모델을 통한 각 메일의 요약, 키워드 추출
4. 처리가 완료된 후 기본 웹 브라우저가 조용히 열리며 완성된 대시보드를 표시

## 5. 문제 해결 (Troubleshooting)
- **로딩이 계속되는 문제:** 콘솔 창(터미널)에 표시된 에러 로그를 확인해보세요. API 일일 할당량을 다 사용했거나 키가 잘못된 경우가 많습니다.
- **로그인 실패(Login Failed):** `config.json` 의 아이디/앱 비밀번호를 다시 확인하세요. 띄어쓰기가 들어갔는지 점검해 보세요.
